# Security Policy

Please report suspected vulnerabilities privately to `contact@projectmonet.space`. Do not open a public issue containing exploit details, personal information, credentials, or form-submission data.

Include the affected URL, a concise description, reproduction steps, and the potential impact. Project Monet will acknowledge legitimate reports when practical. This policy does not create a bug-bounty program or authorize destructive testing, denial-of-service testing, social engineering, or access to other people's data.
